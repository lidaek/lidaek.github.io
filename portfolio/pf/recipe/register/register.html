레시피(Recipe)란?

특정 "조건"이 발생될 때 원하는 동작을
"실행"하도록 만들어 놓은 지시문입니다.

1. 조건 채널선택

조건이 발생될 곳을 선택합니다.

2. 조건 계정선택

채널에 사용할 계정을 연결후 선택합니다.
계정이 필요 없는 채널은 이 단계가 생략될 수 있습니다.

3. 조건 선택

원하는 동작을 위하여 사전에 필요한 조건이 무엇인지 선택합니다.

4. 필터

필터를 설정할 경우, 설정에 맞을 때에만 조건이 발생된 것으로 처리됩니다. 웹 사이트에서 어떤 내용을 검색할 때 원하는 결과를 얻기 위해 검색 조건을 강화하는 과정과 동일합니다.

5. 실행 채널선택

원하는 동작이 실행될 곳을 선택합니다.

6. 실행 계정

채널에 사용할 계정을 연결후 선택합니다.
계정이 필요 없는 채널은 이 단계가 생략될 수 있습니다.

7. 실행 선택

해당 채널에서 실행 할 내용이 무엇인지 선택합니다.

필터 설정 도움말

1. 설정추가 (AND)

해당 설정까지 모두 부합되는 데이터가
있을 때만
조건이 발생한 것으로 처리

설정A : 다리가 4개인 동물 토끼, 거북이, 개 설정B : 종류가 파충류인 동물 거북이, 뱀 결과 C : 설정A와 설정B를 모두 만족하는 것은 거북이 입니다.
2. 그룹추가 (OR)

각 그룹 중에서 부합된 결과가
단 하나라도 있을 때
조건 발생으로 처리

그룹A : 다리가 4개인 동물 토끼, 거북이, 개 그룹B : 종류가 파충류인 동물 거북이, 뱀 결과 C : 그룹A 또는 그룹B에 해당하는 것은 토끼, 거북이, 개, 뱀 입니다.
참고사항
  • 필수 조건을 만족한 데이터에 필터 설정이 적용됩니다.
  • 그룹 내의 항목간에도 1번과 같이 AND 조건을 추가할 수 있습니다.
